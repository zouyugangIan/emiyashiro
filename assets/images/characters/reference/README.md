# HF Shirou Attack References

These images are design references. Runtime combat still uses transparent 256px-frame sprite sheets from `assets/images/characters`, while reference module mode samples the boards as grid atlases for in-game attack previews.

## Runtime Sheets

- `hf_shirou_overedge_light_combo_sheet.png`: 11 frames. `V` uses frames 0-2 for release startup, light combo uses frames 2-4, 5-7, and 8-10.
- `hf_shirou_overedge_heavy_combo_sheet.png`: 17 frames. `K` plays the full heavy combo sheet.

- `hf_shirou_attack_heavy_reference.png`: heavy attack rows.

- `hf_shirou_advanced_attack_modules_overview.png`: advanced combo and ultimate overview.

## Runtime Mapping

- `Shift+V`: enables the reference module mode and shows the overview board.
- `J/Z/L/U`: ground light board, or air combo board while airborne, or mobility board while crouching.
- `K`: heavy board, or ultimate board while crouching.
- `X`: ninjutsu/projectile board, or weapon projection board while crouching.
